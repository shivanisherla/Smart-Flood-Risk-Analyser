#  Smart Flood Risk Analyser

##  Project Overview

The **Smart Flood Risk Analyser** is an intelligent system designed to predict and analyze flood-prone areas using **Machine Learning (ML)** and **Geographic Information System (GIS)** techniques.

It processes environmental data such as rainfall, elevation, land use, and drainage patterns to classify regions into **Low, Medium, and High flood risk zones**.

---

##  Objectives

* Analyze environmental data for flood prediction
* Classify regions into flood risk categories
* Visualize flood-prone areas using maps
* Provide early warning alerts to users
* Support disaster management and decision-making

---

##  Technologies Used

* **Programming Language:** Python
* **Libraries:** Pandas, Folium, Streamlit
* **Machine Learning Models:**

  * Decision Tree
  * Random Forest
  * Support Vector Machine (SVM)
* **GIS Tool:** QGIS
* **Framework:** Streamlit / Flask
* **Database:** SQLite

---

##  System Architecture

The system follows a pipeline approach:

1. Data Collection (Rainfall, DEM, Land Use, Satellite Data)
2. Data Preprocessing
3. Feature Extraction
4. Machine Learning Model Training
5. Flood Risk Prediction
6. Risk Classification (Low / Medium / High)
7. Visualization (Maps & Dashboard)
8. Alert Notification System

---

##  Features

*  Flood risk prediction using ML models
*  Interactive flood risk maps
*  Performance analysis of algorithms
*  Email alert system for high-risk areas
*  User login & registration system

---

##  Dataset

* Satellite data from Landsat & Sentinel
* Environmental data (rainfall, elevation, slope, etc.)
* Village dataset (latitude, longitude, risk levels)

##  Algorithms Comparison

| Algorithm     | Accuracy | Advantage                           |
| ------------- | -------- | ----------------------------------- |

##  How to Run the Project

1. Clone the repository
2. Install required libraries:

   ```bash
   pip install -r requirements.txt
   ```
3. Run the application:

   ```bash
   streamlit run app.py
   ```
4. Open in browser and explore the dashboard
